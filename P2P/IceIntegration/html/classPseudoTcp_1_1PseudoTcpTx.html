Detailed Description

Encapsulates the sending half of the PseudoTcp protocol.

The TX part is responsable for:

  1. Sending Messages, assigning Message number
  2. Keeping track of which Messages have been acked
  3. Keeping track of overdue acks
  4. Resending Messages when necessary
