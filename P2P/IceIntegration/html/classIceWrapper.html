Detailed Description

The IceWrapper class furnishes a C++ interface to the low-level C functionality provided by the p2pConnApi.c and rsComm.c files. IceWrapper provides a facade to C++ for invoking the low-level ice functions. It uses static functions for the callbacks.

For each low-level connection (P2pConnection) created, IceWrapper creates an IceConnection and a MessageDispatcher. It keeps two maps which relate the userName to the relative IceConnection and MessageDispatcher.

IceWrapper also supplies a function which scans all existing MessageDispatchers to see if any have input waiting.
