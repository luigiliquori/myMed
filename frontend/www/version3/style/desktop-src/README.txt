
Installation de lesscss
Télécharger et installer Node.js (http://nodejs.org/#download)
./configure
make
make install

Télécharger https://github.com/cloudhead/less.js/tarball/master
Ajouter less/bin dans $PATH

utilisez la commande comme ceci :
	Afficher le fichier généré
	>lessc file.less
	Compiler le fichier dans un fichier CSS
	>lessc file.less file.css

./configure

Add less/bin in your $PATH var
